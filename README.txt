# Aplicación SWAPI

Esta aplicación (fichero.py) permite descargar y gestionar datos de personajes, planetas y 
vehículos del universo de Star Wars utilizando la [API de SWAPI](https://swapi.dev/). 
Los usuarios pueden buscar información específica a través de un menú interactivo.

Además conseguí realizar una GUI conel fichero prueba.py, con las mismas funcionalidades. 

## Funcionalidades

1. **Descarga de Datos**: La aplicación descarga automáticamente datos sobre personajes, planetas y vehículos desde la API de SWAPI.
2. **Búsqueda**: Permite buscar personajes, planetas y vehículos por nombre.
3. **Interfaz de Usuario**: Utiliza un menú de texto simple para interactuar con el usuario.

## Requisitos

- Python
- Librería `requests`
